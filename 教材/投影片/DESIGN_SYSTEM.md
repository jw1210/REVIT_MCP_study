# AI x Revit 投影片設計規範

## 視覺設計系統 (Design System)

### 🎨 主色調

| 用途 | 顏色 | Hex |
|:----|:----|:----|
| 主色 | 深藍 | #1E3A5F |
| 輔色 | 淺藍 | #4A90D9 |
| 強調色 | 金色 | #D4A84B |
| 背景 | 淺灰白 | #F5F7FA |
| 文字 | 深灰 | #2D3436 |

---

### 📐 統一風格 Prompt（所有圖片必須使用）

**Base Style Prompt（基礎風格前綴）**：

```
Minimalist isometric illustration style, clean geometric shapes, 
soft shadows, limited color palette of navy blue (#1E3A5F), 
light blue (#4A90D9), gold accent (#D4A84B) on light gray-white 
background (#F5F7FA). Professional architecture industry aesthetic, 
suitable for corporate presentation. Consistent line weight, 
no gradients, flat design with subtle depth. No text or labels.
```

---

### 🖼️ 版面規格

| 元素 | 規格 |
|:----|:----|
| 尺寸 | 16:9 比例 |
| 主視覺區 | 佔版面 70% |
| 留白 | 四周保留 10% 邊距 |
| 圖示風格 | 等軸測圖 (Isometric) |
| 人物 | 簡化幾何人形，無細節面部 |
| 建築 | 線條式建築圖，藍圖風格 |

### 📝 完整 Prompt 模板

生成每張投影片圖片時，使用以下格式：

```
[Base Style Prompt] + [具體內容描述]
```

**範例**：

```
Minimalist isometric illustration style, clean geometric shapes, 
soft shadows, limited color palette of navy blue (#1E3A5F), 
light blue (#4A90D9), gold accent (#D4A84B) on light gray-white 
background (#F5F7FA). Professional architecture industry aesthetic, 
suitable for corporate presentation. Consistent line weight, 
no gradients, flat design with subtle depth. No text or labels.

[具體內容]: Three-tier vertical structure showing: top - simplified 
person at desk representing client; middle - office building icon 
representing server; bottom - construction crane representing Revit. 
Connected by flowing arrows.
```
