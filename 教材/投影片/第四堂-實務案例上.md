## Slide 2: 防火檢查對話示範

**標題**：跟著我說

---

### 📝 講師備忘錄

**現場操作示範（15 分鐘）**：

帶學員一起輸入以下對話：

---

**步驟 1：取得樓層**
```
你：請列出這個專案的所有樓層
```
> AI 會回傳樓層清單

---

**步驟 2：查詢牆體**
```
你：請查詢 1F 的所有牆，告訴我每面牆的防火時效
```
> AI 會列出每面牆的防火參數

---

**步驟 3：標記不合格**
```
你：把防火時效不足 1 小時的牆標記為紅色，
達 2 小時以上的標為綠色
```
> AI 會批次上色

---

**步驟 4：產生報告**
```
你：請產生一份防火檢查報告
```
> AI 會產生整理好的報告

---

**顏色方案說明**：

| 防火時效 | 顏色 | RGB |
|:--------|:-----|:----|
| ≥ 2 hr | 綠色 | (0, 180, 0) |
| ≥ 1 hr | 黃色 | (255, 255, 0) |
| < 1 hr | 紅色 | (255, 0, 0) |

## Slide 3: 案例二 - 容積檢討

![容積計算圖示](./images/unified_08_floor_area_1767752247292.png)

**標題**：自動計算樓地板面積

---

### 📝 講師備忘錄

**案例背景（2 分鐘）**：

> 「這張圖很清楚：多層建築的爆炸圖，每層樓面積分開計算。
> 
> 旁邊有計算機、長條圖、圓餅圖——這就是我們要產出的報告。」

**容積計算規則複習**：

| 空間類型 | 計入容積 |
|:--------|:-------:|
| 主要空間（客廳、臥室） | 100% |
| 陽台 | 50% |
| 樓梯間、機房 | 0% |

---

## Slide 4: 容積檢討對話示範

**標題**：三步驟完成容積計算

---

### 📝 講師備忘錄

**現場操作示範（15 分鐘）**：

---

**步驟 1：取得房間資料**
```
你：請列出每層樓的所有房間和面積
```

---

**步驟 2：分類計算**
```
你：請根據以下規則計算容積：
- 主要空間計入 100%
- 陽台計入 50%
- 樓梯間、電梯間不計入
```

---

**步驟 3：對照法規**
```
你：這是住宅區，容積率上限 300%，
基地面積 500 平方公尺，請判斷是否符合
```

---

**報告範例**：
```
┌────────────────────────────┐
│      容積檢討報告           │
├────────────────────────────┤
│  一樓：450 ㎡               │
│  二樓：420 ㎡               │
│  三樓：380 ㎡               │
│  ────────────              │
│  總計：1,250 ㎡             │
│  基地面積：500 ㎡           │
│  容積率：250%              │
│  法規上限：300%            │
│  ────────────              │
│  結果：✅ 符合法規          │
└────────────────────────────┘
```
