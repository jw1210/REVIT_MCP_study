## Slide 4: 上色 5 步驟

**標題**：避免接合問題的標準流程

---

### 📝 講師備忘錄

**這是重點！（10 分鐘）**

```
步驟 1: 清除舊顏色
    ↓
步驟 2: 取消牆柱接合
    ↓
步驟 3: 牆體上色
    ↓
步驟 4: 柱子上色（黑色）
    ↓
步驟 5: 恢復牆柱接合
```

---

**步驟 1：清除舊顏色**
```
你：請清除當前視圖中所有元素的顏色覆寫
```

> 「為什麼要先清除？因為舊的顏色可能會干擾新的上色。」

---

**步驟 2：取消牆柱接合**
```
你：請取消當前視圖中牆和柱子的接合
```

> 「這一步很關鍵！如果牆柱接合，上色時顏色可能會『漏』到別的元素上。」

---

**步驟 3：牆體上色**
```
你：請根據「防火性能」參數對牆體上色：
- 值為 2 → 綠色
- 值為 1 → 黃色
- 沒有值 → 紫色
```

---

**步驟 4：柱子上色**
```
你：請把所有柱子標記為黑色
```

> 「柱子統一用黑色，讓它們不會和牆的顏色混淆。」

---

**步驟 5：恢復牆柱接合**
```
你：請恢復牆和柱子的接合
```

> 「完成上色後，要把接合恢復回來，不然模型會怪怪的。」

## Slide 5: 顏色方案參考

**標題**：常用顏色搭配

---

### 📝 講師備忘錄

**顏色建議（3 分鐘）**：

| 顏色 | RGB | 用途 |
|:----|:---:|:----|
| 🔴 紅 | (255, 0, 0) | 不合格、警告 |
| 🟡 黃 | (255, 255, 0) | 注意、待確認 |
| 🟢 綠 | (0, 180, 0) | 合格、通過 |
| 🟣 紫 | (200, 0, 200) | 未設定、空值 |
| ⚫ 黑 | (30, 30, 30) | 柱子、結構 |

**透明度設定**：

| 透明度 | 效果 |
|:-----:|:-----|
| 0 | 完全不透明 |
| 30 | 輕微透明 |
| 50 | 半透明 |
