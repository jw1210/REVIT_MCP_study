## 🔥 案例一：防火等級檢查

### 情境說明

你的案子即將送審，需要檢查所有牆體的防火時效是否符合法規。
傳統做法是逐一點選牆體查看屬性，現在讓 AI 幫你自動完成。

### 學習目標

- ✅ 使用自然語言查詢牆體資訊
- ✅ 根據參數值對元素上色
- ✅ 產生檢查報告

### 操作步驟

#### 步驟 1：確認 MCP 服務已啟動

在 Revit 的「MCP Tools」面板，確認服務已開啟。

#### 步驟 2：取得樓層清單

向 AI 說：
```
請列出這個專案的所有樓層
```

AI 回應範例：
```
這個專案有以下樓層：
- 1F（高程：0m）
- 2F（高程：3.6m）
- 3F（高程：7.2m）
```

#### 步驟 3：查詢外牆資訊

向 AI 說：
```
請幫我查詢 1F 的所有牆，並告訴我每面牆的防火時效
```

AI 會篩選牆體並回傳防火時效參數。

#### 步驟 4：標記不合格元素

向 AI 說：
```
把所有防火時效不足 1 小時的牆標記為紅色，
防火時效達 2 小時以上的標記為綠色
```

AI 會批次處理並上色：

| 防火時效 | 顏色 | 意義 |
|:--------|:-----|:-----|
| ≥ 2 小時 | 🟢 綠色 RGB(0, 180, 0) | 合格 |
| ≥ 1 小時 | 🟡 黃色 RGB(255, 255, 0) | 注意 |
| < 1 小時 | 🔴 紅色 RGB(255, 0, 0) | 不合格 |

#### 步驟 5：產生檢查報告

向 AI 說：
```
請產生一份防火等級檢查報告
```

報告範例：
```
┌────────────────────────────────────────────┐
│            防火等級檢查報告                 │
├────────────────────────────────────────────┤
│  檢查日期：2026-01-07                       │
│  總牆數：48 面                              │
│  合格：42 面                                │
│  不合格：6 面                               │
├────────────────────────────────────────────┤
│  不合格清單：                               │
│  ├─ 牆 ID 12345：1F 隔間牆，時效 30 分鐘    │
│  ├─ 牆 ID 12346：2F 走廊牆，時效 0 分鐘     │
│  └─ ...                                    │
├────────────────────────────────────────────┤
│  建議：                                     │
│  1. 調整牆體類型或材質                       │
│  2. 增加防火塗料                            │
└────────────────────────────────────────────┘
```

### 法規參考

| 建築類型 | 外牆防火時效 | 內牆防火時效 |
|:--------|:----------:|:----------:|
| 住宅（6層以下） | 1 小時 | 30 分鐘 |
| 住宅（7層以上） | 2 小時 | 1 小時 |
| 辦公大樓 | 2 小時 | 1 小時 |
| 商場 | 2 小時 | 1 小時 |

## 📐 案例二：容積檢討

### 情境說明

設計完成後需要確認容積率是否符合法規，
AI 可以幫你快速計算各樓層面積並產生報告。

### 學習目標

- ✅ 取得房間面積資料
- ✅ 分類計算（主空間、陽台、附屬）
- ✅ 對照法規上限

### 操作步驟

#### 步驟 1：取得房間資料

向 AI 說：
```
請列出每層樓的所有房間和面積
```

AI 會逐層回傳房間清單。

#### 步驟 2：分類計算

向 AI 說：
```
請根據以下規則計算容積：
- 主要空間（住宅、辦公）計入 100%
- 陽台計入 50%
- 樓梯間、電梯間不計入
```

#### 步驟 3：對照法規

向 AI 說：
```
這是住宅區，容積率上限是 300%，
基地面積是 500 平方公尺，請判斷是否符合
```

AI 會自動計算並判斷。

### 輸出報告範例

```
┌────────────────────────────────┐
│        容積檢討報告             │
├────────────────────────────────┤
│  一樓：450 ㎡                   │
│  二樓：420 ㎡                   │
│  三樓：380 ㎡                   │
│  ──────────────────            │
│  總計：1,250 ㎡                 │
│  基地面積：500 ㎡               │
│  容積率：250%                   │
│  法規上限：300%                  │
│  ──────────────────            │
│  結果：✅ 符合法規               │
└────────────────────────────────┘
```

### 分類規則參考

| 類型 | 計入容積 | 範例空間 |
|:----|:-------:|:--------|
| 主要空間 | 100% | 客廳、臥室、辦公室 |
| 陽台 | 50% | 陽台、露台 |
| 附屬空間 | 0% | 樓梯間、電梯間、機房 |

## 🚶 案例三：走廊寬度分析

### 情境說明

建築技術規則對走廊寬度有最低要求，
AI 可以自動識別走廊並檢查是否符合法規。

### 學習目標

- ✅ 識別走廊空間
- ✅ 分析淨寬是否符合法規
- ✅ 自動建立尺寸標註

### 操作步驟

#### 步驟 1：識別走廊

向 AI 說：
```
請找出 2F 的所有走廊
```

> 💡 AI 會自動識別名稱包含「走廊」「Corridor」「廊道」「通道」的空間

#### 步驟 2：分析寬度

向 AI 說：
```
請分析這些走廊的寬度是否符合建築技術規則
（住宅 ≥ 1.2m，公共建築 ≥ 1.6m）
```

AI 會逐一分析並回報：
```
走廊分析結果：
- 走廊A：寬度 1.5m ✅ 符合
- 走廊B：寬度 1.1m ❌ 不足（需 ≥ 1.2m）
- 走廊C：寬度 1.8m ✅ 符合
```

#### 步驟 3：建立尺寸標註

向 AI 說：
```
請在不合格的走廊建立寬度尺寸標註
```

AI 會自動在平面圖上建立標註。

### 法規標準

| 建築類型 | 走廊最小淨寬 |
|:--------|:----------:|
| 住宅 | 1.2m |
| 辦公 | 1.2m |
| 學校 | 2.4m（單側教室）/ 3.6m（雙側教室） |
| 醫院 | 2.4m |

## 🎨 案例四：元素上色工作流程

### 情境說明

當需要在圖面上用顏色區分不同類型的元素時，
有一套標準的操作流程確保結果正確。

### 學習目標

- ✅ 理解視圖類型與樣式的關係
- ✅ 執行標準 5 步驟上色流程
- ✅ 清除顏色還原模型

### 執行前確認

| 確認項目 | 選擇 |
|:--------|:----|
| **視圖類型** | 平面圖 → 切割樣式 / 立面圖 → 表面樣式 |
| **分類參數** | 例如：`s_CW_防火防煙性能` |
| **顏色方案** | 每個參數值對應什麼顏色？ |

### 標準 5 步驟流程

```
步驟 1               步驟 2               步驟 3
清除舊顏色    ──▶    取消牆柱接合    ──▶    牆體上色
     │                    │                    │
     ▼                    ▼                    ▼
步驟 5               步驟 4               
恢復牆柱接合  ◀──    柱子上色（黑色）
```

#### 詳細操作

**步驟 1：清除舊顏色**
```
請清除當前視圖中所有元素的顏色覆寫
```

**步驟 2：取消牆柱接合**

> 💡 這一步很重要！牆和柱子接合時，上色可能會有問題

```
請取消當前視圖中牆和柱子的接合
```

**步驟 3：牆體上色**
```
請根據「防火性能」參數對牆體上色：
- 值為 2 → 綠色
- 值為 1 → 黃色
- 沒有值 → 紫色
```

**步驟 4：柱子上色**
```
請把所有柱子標記為黑色
```

**步驟 5：恢復牆柱接合**
```
請恢復牆和柱子的接合
```

### 顏色方案範例

| 參數值 | 顏色 | RGB |
|:-------|:-----|:----|
| 2（高防火） | 綠色 | (0, 180, 0) |
| 1（一般防火） | 黃色 | (255, 255, 0) |
| 空值（未設定） | 紫色 | (200, 0, 200) |
| 柱子 | 黑色 | (30, 30, 30) |

### 還原操作

完成檢視後，要還原模型：
```
請清除當前視圖的所有顏色覆寫
```
