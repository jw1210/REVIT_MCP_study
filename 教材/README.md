# AI x Revit 建築實務教材

## 給建築設計師的 24 小時深度課程

---

> **課程目標**：讓純 Revit 使用者學會運用「提示工程」與 AI 協作，
> 開發 BIM 工作流程、工具和知識庫。
>
> **核心理念**：不是教你寫程式，而是教你如何用 Prompt 和 AI 一起開發！
